# EasyFuse 

EsayFuse is a pipline for fusion gene detection from RNA-seq data.


## Installation

### Dependencies

 - R (>= 3.5.1)
 - R packages: 
 
    - optparse
    - tidyverse
    - randomForest

  Install packages within R by
  
  ```
  install.packages(c("optparse", "tidyverse", "randomForest"))
  ```
